ได้รับเงินเมื่อแจ้งความเปลี่ยนแปลง (pull request)

เพื่อส่งเสริมความโปร่งใส และปรับปรุงประสิทธิภาพและฟังก์ชันการทำงานของเว็บไซต์ของเรา Binary.com ได้โอเพ่นซอสส่วนหน้าทั้งหมดของ codebase ของเรา และตอนนี้เรากำลังส่งเสริมชุมชนให้สนับสนุนช่วยเหลือในการพัฒนาดังกล่าว

เราจะจ่ายเงินให้กับการแจ้งความเปลี่ยนแปลงที่ได้รับการยอมรับและรวมเข้ากับ codebase หลัก

มีโครงการใดอยู่บ้าง?

ทั้ง Binary.com แพลตฟอร์มส่วนหน้าของผู้ใช้งานและไลบราลีทั้งหมด มีสิทธิ์

จะส่งแจ้งความเปลี่ยนแปลง (pull request) อย่างไร

  • แยกที่เก็บของ Github ที่เกี่ยวข้อง
  • ทำการเปลี่ยนแปลงรหัสในส่่วนแยกของคุณ
  • ส่งการแจ้งความเปลี่ยนแปลงไปยังตัวหลักในสถานที่เก็บเริ่มแรกของ GitHub
  • ส่งอีเมลไป pull-requests@binary.com เพื่อขอให้ทีมพัฒนาของเราพิจารณาการแจ้งความเปลี่ยนแปลง (pull request)

ฉันจะทดสอบการเปลี่ยนแปลงต่างๆ ได้อย่างไร?

โครงการส่วนหน้าทั้งหมดของ Binary.comถูกจัดให้ใช้งานโดยใช้พื้นที่หน้าของ Github กรุณาทำความคุ้นเคยกับพื้นที่นั้น และอ่านคำแนะนำในการใช้งานของสถานที่เก็บแต่ละอันที่เกี่ยวข้อง

ฉันสามารถส่งการเปลี่ยนแปลงประเภทใดบ้าง?

  • คุณลักษณะใหม่
  • เครื่องมือในการทำกราฟดีกว่า
  • ตัวชี้วัดทางเทคนิคใหม่
  • การใช้งานที่ปรับปรุงดีขึ้น
  • ความเข้ากันได้เบราว์เซอร์ที่ปรับปรุงดีขึ้น
  • การตอบสนองที่ปรับปรุงดีขึ้น
  • แสดงผลเร็วกว่า
  • แก้ไขข้อผิดพลาด
  • การจัดเตรียมเอกสารที่ปรับปรุงดีขึ้น
  • ความครอบคลุมการทดสอบที่ดีกว่า

ฉันสามารถเขียนหน้ากากใหม่จากศูนย์ได้ไหม?

แน่ใจ เรายินดีต้อนรับโครงการ เช่นแอพสำหรับอุปกรณ์เคลื่อนที่ใหม่ แอพ chatbot ส่วนขยายเบราว์เซอร์ เครื่องมือสำหรับกราฟใหม่

สามารถส่งช่องโหว่ด้านความปลอดภัยได้หรือไม่

ใช่ แต่เราต้องการให้ท่านส่งรายการช่องโหว่ของความปลอดภัยผ่านโปรแกรม Bug Bounty ของเรา

ฉันช่วยพัฒนาระบบส่วนหลังบ้านได้ไหม?

รหัสส่วนหลังเราส่วนใหญ่อยู่ใน Perl และสามารถค้นพบผลงานโอเพ่นซอสของเราที่CPAN. สามารถพบซอสโค้ดที่สอดคล้องกันได้ในGสถานที่เก็บของ Githubซึ่งขึ้นต้นด้วยคำว่า "perl-" เรายินดีที่จะรับผลงานที่จะทำในสถานที่เก็บนี้ เพื่อปรับปรุงการจัดเก็บเอกสารและครอบคลุมการทดสอบจนเป็นที่สังเกต

ฉันจะได้รับชำระเงินอย่างไร?

เราจะจ่ายให้แก่การแจ้งการเปลี่ยนแปลงที่ได้รับการอนุมัติจากทีมพัฒนาของเราและได้รวมเข้าไปในตัวหลักเท่านั้น ถ้าทีมพัฒนาอนุมัติการแจ้งความเปลี่ยนแปลงของคุณ เราจะจ่ายคุณเป็นเงินดิจิทัล คุณสามารถเลือกบิทคอยน์ ไลท์คอยน์ หรืออีเธอร์ก็ได้

สิ่งที่ฉันคาดหวังว่าจะได้รับ?

ทีมพัฒนาของเราจะพิจารณาว่าคุณได้รับเงินเท่าไหร่ คุณจะได้รับเงินตามในช่วงต่อไปนี้:

  • การเปลี่ยนแปลงขนาดเล็ก – โดยทั่วไป $5-20
  • การเปลี่ยนแปลงขนาดเล็ก – โดยทั่วไป 20-50 เหรียญสหรัฐ
  • ค่าเฉลี่ยการเปลี่ยนแปลงโดยทั่วไป คือ 50 - 250 เหรียญสหรัฐ
  • ค่ากลางการเปลี่ยนแปลงโดยทั่วไป คือ 250 - 750 เหรียญสหรัฐ
  • การเปลี่ยนแปลงขนาดใหญ่ – โดยทั่วไป $750-2,500
  • การเปลี่ยนแปลงมาก/โครงการใหม่ทั้งหมด - $2,500-10,000

ฉันยังคงมีกรรมสิทธิ์ในโปรแกรมหรือไม่?

ไม่ เมื่อข้อมูลแจ้งความเปลี่ยนแปลงของคุณได้ถูกรวมเข้ากับตัวหลักโดยทีมพัฒนาของเรา และคุณได้รับเงินแล้ว มีผลว่าคุณได้มอบสิทธิ์ของรหัส จากนั้นข้อมูลดังกล่าวจะกลายเป็นส่วนหนึ่งของ codebase Binary.com หลักภายใต้ใบรับรองโอเพ่นซอสของสถานที่เก็บที่เกี่ยวข้อง

ถ้าไม่มีการตกลงรางวัลล่วงหน้า ข้าพเจ้าจะทราบได้อย่างไรว่าได้รับการจ่ายเงินอย่างเป็นธรรม

เราต้องการปรับปรุงแพลตฟอร์มของเราตลอดเวลา ดังนั้น เป็นความต้องการของเราที่จะจ่ายเงินให้แก่ผู้สนับสนุนอย่างเป็นธรรม ถ้าเราไม่ทำ คุณจะหยุดส่งการแจ้งความเปลี่ยนแปลง ซึ่งถือเป็นการทำร้ายโครงการนี้เอง

ฉันสามารถทำงาน Binary.com เต็มเวลาได้หรือไม่?

เรามองหาบุคคลที่มีความสามารถเสมอ กรุณาสมัครผ่านทางหน้างานของเรา

สามารถปรึกษาหารือเรื่อง codebases กับทีมพัฒนาของ Binary.comได้หรือไม่

แน่นอน กรุณาส่งอีเมลไปยังpull-requests@binary.comเพื่อขอให้ส่งคำเชิญเพื่อเข้าสู่ช่อง Slack ของเรา
