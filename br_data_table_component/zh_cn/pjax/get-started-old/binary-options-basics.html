二元期权基础知识

二元期权很容易了解。它被称为“二元”是因为只能有2种结果：赢或输。如果预测正确，您会获得交易开始时设定的赔付额。不然，您将损失初始资金。

您只需要做四个决定，即可交易二元期权合约。

1. 选择标的资产

首先选择您要交易的资产，如黄金、原油、股票或外汇（二元期权的价值基于标的资产价格）。期权交易一个很大的优势就是您并非买卖实际资产。

2. 选择交易期限

每个二元期权合约都有固定期限- 在 Binary.com，您可选择从10秒至365天之间的任何期限。

3. 选择如何进行市场交易

Binary.com 为每一资产提供以下四种交易方式：

  • 上涨/下跌 - 预测市价是否从目前价位上升或下跌。
  • “高于/低于” - 预测到期时市场价格是高于还是低于目标价格。
  • 触及/未触及 - 预测市价是否触及或不触及一个目标价格。
  • “范围之内/之外” - 预测市价是否保持在两个目标价格的范围之内，或超出范围之外。

4. 选择您的投注额与潜在回报率

确定赔付额，如果您的预测正确的话，这就是您的潜在回报率。我们的系统将进一步计算您的投注额，即购入合约的总成本。Binary.com提供从1美元到5万美元的赔付额。
