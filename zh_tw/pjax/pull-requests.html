因拉動請求獲得報酬

為了促進我們網站的透明度和提高功能與性能，Binary.com 的整個代碼庫前端已提供開源，我們現在也正在積極鼓勵社會對其發展作出貢獻。

我們會對任何被接受並合併到主代碼庫的拉動請求提供報酬。

哪些專案是可用的？

所有 Binary.com 前端平台和程式庫 都有資格。

如何提交拉動請求？

  • 對有關 Github 存儲庫進行分叉
  • 在分叉中更改代碼
  • 提交拉動請求到 GitHub 原始存儲庫中的主程式
  • 發送電子郵件到 pull-requests@binary.com 以便要求我們的開發團隊審查您的拉動請求

如何測試更改？

所有 Binary.com 前端專案都部署使用 Github 頁設施。請熟悉此設施的使用方法，並讀取每個相關存儲庫中的部署說明。

我可以提交什麼樣的變化？

  • 新功能
  • 更好的圖表設施
  • 新的技術指標
  • 已提高的可用性
  • 已改進的瀏覽器相容性
  • 已改進的回應能力
  • 更快呈現
  • Bug 修復
  • 已改進的文檔
  • 更好的測試覆蓋率

我可以從頭開始寫新前端程式嗎？

確定。我們歡迎新手機應用、聊天機器人應用、瀏覽器延伸軟體和新的圖表工具等專案。

我可以提交安全性漏洞嗎？

是的。但我們寧願您通過我們的bug賞金項目提交安全漏洞。

我可以提供後端代碼嗎？

我們的後端代碼主要是在 Perl 中，而我們的開源貢獻可能會在CPAN出現。對應的原始程式碼會在以首碼"perl-"開頭的Github資料庫 中出現。我們歡迎大家對這些資料庫提供貢獻，特別是經過改進的文檔和測試覆蓋率。

如何獲得報酬？

我們只對經過開發團隊批准並合併到主程式的拉動請求給予報酬。如果開發團隊批准您的拉動請求，我們會付加密貨幣給您。您可以選擇比特幣、萊特幣或以太幣。

我可以獲得什麼報酬？

我們的開發團隊將決定您可以得到多少報酬。您可以期待以下的報酬範圍：

  • 極小量變化 —一般為 5-20 美元
  • 小量變化 —一般為 20-50 美元
  • 平均大小的變化 — 一般為 50-250 美元
  • 中等變化 —一般為 250-750 美元
  • 大變化 — 一般為 750-2,500 美元
  • 非常大的變化/整個新專案 — 2,500 -10,000 美元

我是否保留代碼的擁有權？

不。一旦我們的開發團隊將您的拉動請求合併到主程式內，以及您已獲取報酬後，表示您已有效交付代碼的權利。它將按照有關存儲庫的開放源許可證成為主程式Binary.com 代碼庫的一部分 。

如果獎勵沒事先商定，我怎麼知道所獲報酬是合理的？

我們要不斷改進我們的平台，所以，給予貢獻者合理的報酬符合我們的利益。如果我們不這樣做，您就會停止貢獻拉動請求，導致這個項目弄巧成拙。

我可否為 Binary.com 全職工作？

我們一直都在尋找人才 — 請通過我們的 職業 頁面提出申請。

我可以與 Binary.com 開發團隊討論基本代碼嗎？

答案是肯定的！請發送電子郵件至 pull-requests@binary.com，請求受邀前往slack通道。
