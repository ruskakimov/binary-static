因拉动请求获得报酬

为了促进我们网站的透明度和提高功能与性能，Binary.com 的整个代码库前端已提供开源，我们现在也正在积极鼓励社会对其发展作出贡献。

我们会对任何被接受并合并到主代码库的拉动请求提供报酬。

哪些项目是可用的？

所有 Binary.com 前端平台和库 都有资格。

如何提交拉动请求？

  • 对有关 Github 存储库进行分叉
  • 在分叉中更改代码
  • 提交拉动请求到 GitHub 原始存储库中的主程序
  • 发送电子邮件到 pull-requests@binary.com 以便要求我们的开发团队审查您的拉动请求

如何测试更改？

所有 Binary.com 前端项目都部署使用 Github 页设施。请熟悉此设施的使用方法，并读取每个相关存储库中的部署说明。

我可以提交什么样的变化？

  • 新功能
  • 更好的图表设施
  • 新的技术指标
  • 已提高的可用性
  • 已改进的浏览器兼容性
  • 已改进的响应能力
  • 更快呈现
  • Bug 修复
  • 已改进的文档
  • 更好的测试覆盖率

我可以从头开始写新前端程序吗？

确定。我们欢迎新手机应用、聊天机器人应用、浏览器扩展软件和新的图表工具等项目。

我可以提交安全漏洞吗？

是的。但我们宁愿您通过我们的bug赏金项目提交安全漏洞。

我可以提供后端代码吗？

我们的后端代码主要是在 Perl 中，而我们的开源贡献可能会在 CPAN出现 。对应的源代码会在以前缀"perl-"开头的 Github 资料库中出现。我们欢迎大家对这些资料库提供贡献，特别是经过改进的文档和测试覆盖率。

如何获得报酬？

我们只对经过开发团队批准并合并到主程序的拉动请求给予报酬。如果开发团队批准您的拉动请求，我们会付加密货币给您。您可以选择比特币、莱特币或以太币。

我可以获得什么报酬？

我们的开发团队将决定您可以得到多少报酬。您可以期待以下的报酬范围：

  • 极小量变化 —一般为 5-20 美元
  • 小量变化 —一般为 20-50 美元
  • 平均大小的变化 — 一般为 50-250 美元
  • 中等变化 — 一般为 250-750 美元
  • 大变化 — 一般为 750-2,500 美元
  • 非常大的变化/整个新项目 — 2,500 -10,000 美元

我是否保留代码的所有权？

不。一旦我们的开发团队将您的拉动请求合并到主程序内，以及您已获取报酬后，表示您已有效交付代码的权利。它将按照有关存储库的开放源许可证成为主程序Binary.com 代码库的一部分 。

如果奖励没事先商定，我怎么知道所获报酬是合理的？

我们要不断改进我们的平台，所以，给予贡献者合理的报酬符合我们的利益。如果我们不这样做，您就会停止贡献拉动请求，导致这个项目弄巧成拙。

我可否为 Binary.com 全职工作？

我们一直都在寻找人才 — 请通过我们的 职业 的页面提出申请。

我可以与 Binary.com 开发团队讨论基本代码吗？

答案是肯定的！请发送电子邮件至 pull-requests@binary.com，请求受邀前往slack通道。
